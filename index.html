Advanced Audio Sampler

Designed and built an audio sampler which allows a user to record through a line in or built in mic. The user can then view the waveform of the recorded digital audio on an LCD screen in order to slice it up and map it to a physical pad on the sampler. The user can then playback and apply sound effects to their song pattern.
